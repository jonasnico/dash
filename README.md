# A little dashboard for myself, testing the Neobrutalism component library

## Tech used

- React 19 + TS
- Vite
- Tailwind CSS with [https://www.neobrutalism.dev/](neobrutalism) design system
- shadcn/ui components
- Lucide React icons

## Development

```bash
npm install
npm run dev
```

## Deployment

```bash
npm run build
```
